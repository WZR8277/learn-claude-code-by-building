# s07 Reflection

我的感受是，这一章是比较常规的渐进式加载 skill：先让模型看到有哪些技能，再在真正需要的时候加载完整内容。机制本身不复杂，也没有特别反直觉的地方。

但它让我更注意到另一件事：对开发者来说，skill 本身的设计需要格外讲究技巧。一个 skill 不是简单把文档塞进去，而是要写清楚什么时候该用、解决什么问题、有哪些边界、需要读取哪些额外材料，以及如何避免把无关信息提前灌进上下文。

所以 s07 的重点不只是 `load_skill` 这个工具，而是把“知识入口”做成可发现、可判断、可按需展开的结构。目录层越简洁，正文层越清楚，模型越容易在合适的时候加载合适的知识。
